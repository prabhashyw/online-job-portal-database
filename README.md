# Online Job Portal Database

## Project Description

This project is an Online Job Portal Database developed using Oracle SQL and Oracle APEX. It allows employers to post job vacancies, job seekers to apply for jobs, and manage interview information.

## Technologies Used

- Oracle SQL
- Oracle APEX

## Features

- Manage Job Seekers
- Manage Employers
- Manage Job Vacancies
- Manage Applications
- Manage Interviews

## Database Design

The database is normalised up to Third Normal Form (3NF) to reduce redundancy and improve data integrity.
